# Basic Setup Instructions

## Download and install the repo
npm install

## Add the following libraries

### Next
- npm i -D daisyui@latest
- npm install sharp
- npm install axios
- 
### Python
- pip install flask
- pip install flask-cors
- pip install oracledb
- pip install SQLAlchemy

## Run the Demo
1. npm run dev
2. py app.py
3. run the docker container
